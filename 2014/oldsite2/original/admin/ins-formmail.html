This function allows you to install a form-to-mail script which will enable you to send form requests to an e-mail address.

[Overview] [Fields] [Notes] [Samples] [Errors]


Email: (Required) This field is for the e-mail address to which the form will be sent.

IP: (Required) This field is for the ip of your site (i.e 208.280.23.104).

Domain: (Not Required) This field is for the domain name of your site.

[Overview] [Fields] [Notes] [Samples] [Errors]
DO NOT put the WWW at the beginning of your domain name. (i.e www.yourdomain.com = yourdoamin.com)

[Overview] [Fields] [Notes] [Samples] [Errors]
Invalid email address. - This error means the Email field was left blank. Resubmit the form with an e-mail address.

Invalid IP address - This error means the IP field has invalid characters. Resubmit the form verifying the IP field is correct.

Invalid domain name - This error means the Domain field has invalid characters. Resubmit the form verifying the Domain field is correct.

Cannot open file... - This error means a server side configuration is incorrect. Contact technical support to resolve the error.

Server error - This error means a server side configuration is incorrect. Contact technical support to resolve the error.
