This function allows you to specify a time frame to view your site's error log and cgi error log, so you can debug and fix your site's links and cgi programs.

[Overview] [Fields] [Notes] [Samples] [Errors]
Time Frame: (Required) This field is for the time frame of display error message. The time frame can be Today, Yesterday. This Month or All.

[Overview] [Fields] [Notes] [Samples] [Errors]
You'd better not use "All" time frame at first, Use other time frame instead. If your site has a huge error log messages, it'll take long time for you to display it.
[Overview] [Fields] [Notes] [Samples] [Errors]
Time Frame: 
      
   This Month

[Overview] [Fields] [Notes] [Samples] [Errors]
Invalid Time Frame - This means the you didn't input Time Frame a valid time frame, currently we support only Today, Yesterday, This Month and All.
