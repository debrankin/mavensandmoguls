.htaccess is a function that is use to password-protect web pages. It creates the actual password list along with the specific users and the encrypted passwords. The main purpose of this program is to give the administrator of the web site the option to make his/her site public, where everyone could browse, or private, only specific persons could browsers. This program has four (4) main functions:
  • Create User, where you create users to access your web site.
  • Delete User, where you delete any user that was created to access your web site.
  • Remove File, where you remove the .htpasswd or/and .htaccess file that was created.
  • View File, where you could view all your users in your .htpasswd file.


[Overview] [Fields] [Notes] [Samples] [Errors]
For specific functions, there are specific requirements: Create User
  • User Name: (Required) This field is for identifying the user name you wish to create to access your web site.
  • Password: (Required) This field is for setting the password for the user you are creating to access your web site.
  • Directory: (Required) This field is for specifying which "html" directory you would like to create access for specific users to browser. If Directory is left blank, the htaccess program is created in your "html" directory. Delete User
  • User Name: (Required) This field is for identifying the user name you wish to delete to access your web site.
  • Directory: (Required) This field is for specifying which "html" directory you would like to create access for specific users to browser. If Directory is left blank, the htaccess program is deleted in your "html" directory. Remove File
  • Directory: (Required) This field is for specifying which "html" directory you would like to delete the htaccess program from. View File
  • Directory: (Required) This field is for specifying which .htpasswd file you would like to view. If Directory is left blank, the htaccess program is viewed from your "html" directory, which is the default.
