This function allows you to create and delete frontpage users. You can let your users login your web server using Frontpage Software.

Your created frontpage user only applied for frontpage software, you can't ftp or telnet to your server using this user;

[Overview] [Fields] [Notes] [Samples] [Errors]
Create User: Input the new username you want to create and then you will need to click the radio button choice at the left of Create User.

Delete User: Delete the user you created, you can select from a user list which are created by you. The Delete User field won't appear if you have no users to delete. and you'll need to click the radio choice at the left of Delete User.

Password: (Required) This field is for setting the password for the user you are creating.

Confirm: (Required) This field is for confirming the password.

Role: (Required) This field is for specifying the permssion for your users.

  • Administrator View, add, and change all server content; manage server settings and accounts.
  • Advanced author View, add, and change pages, documents, themes, and borders; recalculate hyperlinks.
  • Author View, add, and change pages and documents.
  • Browser View pages and documents.

[Overview] [Fields] [Notes] [Samples] [Errors]
Invalid user name - This error means that the User Name field was either left blank or contains invalid characters. Resubmit form with an alternate user name.

Invalid password - This error means the Password field was either left blank or contains invalid characters. Resubmit the form with an alternate password.

Incorrect password - This error means the Password and Confirm fields do not match. Resubmit the form.

Duplicate user name - This error means the user name already exists on the server. Resubmit the form using an alternate user name.

Server cannot create user at this time - This error means the server was not available to process the request. If the error persists, contact technical support.

Server error - This means a server side configuration is incorrect. Contact technical support to resolve the error.
