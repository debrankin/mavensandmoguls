This function allows you to change the server admin email address. This email address will be displayed to your browser when there is something wrong with this site, for example, script error or internal server.

[Overview] [Fields] [Notes] [Samples] [Errors]
Server Admin Email: (Required) This field is for the input of new server admin email, your original server admin email will be displayed.

[Overview] [Fields] [Notes] [Samples] [Errors]
Server Admin address will be displayed to your visitor when there is something wrong with your site, for example, script error and internal server, so they can contact you with this address. You can change the email address in the input box.
